RudderStack has launched Lookout, an AI-powered analytics platform that uses an AI agent with access to instrumentation code, tracking plans, warehouse data, and pipeline information to investigate issues and generate insights through natural conversation. Key capabilities include natural language analytics for querying data warehouses without SQL, automated dashboard creation, AI-generated instrumentation code with pull request integration, and automated PR reviews for data contract changes. The platform runs on AWS Bedrock with zero data retention and integrates with Slack and MCP servers.
